# Stock price simulation using random walk
<b> Project Description </b> <br>

This project is about simulating how stock prices move using the Random Walk Model. This model is an idea in finance that says price changes happen randomly and we cannot predict them. <br>

The project uses Python, NumPy and Matplotlib to create possible paths for stock prices and show trends and volatility. <br>

Theory behind the project <br>

Random Walk Theory <br>

Stock prices go up and down randomly <br>

Future prices do not depend on what happened in the past <br>

# Mathematical Model
The price at a time is equal to the previous time multiplied by a random return. <br>

The formula for this is: <br>

Price at time t = Price at time t-1*(1+return at time t) <br>

Where: <br>

Price at time t is the stock price at that time <br>

Return at time t is a random change in the stock price <br>

# Objectives
The goals of this project are to simulate how stock prices behave understand how probability works in finance show many stock price paths and analyze volatility and trends <br>

To achieve these goals the project will use the Random Walk Model to generate many possible stock price paths <br>

The project will also calculate and display the mean trend line and volatility of the stock prices <br>

If the volatility is high the stock prices will fluctuate more <br>
